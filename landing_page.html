Lost@Cruz
Lost at Cruz is a website that helps UCSC students find their lost items by connecting finders and losers through the platform.

Lost at Cruz provides a platform for users to post items they have found or lost. Users who have reported a lost item receive daily notifications about recently found items that may match theirs.

Lost at Cruz is built with React using the Next.js framework for the frontend. For the backend, we use Firebase for database management and authentication. Finally, the website is hosted on Vercel.

Web Crawler
A program that recursively explores all unique internal links starting from a given seed URL, up to a specified depth. It fetches each page, extracts new internal links, and stores each visited page's HTML into a specified directory. Each page is saved with a unique ID, and already visited URLs are skipped to avoid revisiting.
